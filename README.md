# Gyan Kumar Sah - Personal Portfolio

This is the source code for my personal portfolio website, available at [https://www.gyankumarsah.com.np](https://www.gyankumarsah.com.np).

## Overview

The portfolio showcases my research experience, industry experience, academic projects, and software/hackathon projects. It is built using HTML, CSS, and JavaScript.

## Technologies Used

- Vanilla HTML, CSS, and JavaScript
- Vercel for deployment

## Local Development

1. Clone the repository
2. Open `index.html` in your browser, or run a local server (e.g., using `npx serve` or Live Server in VS Code)
